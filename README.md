# Data Incubator - Hospital Compare App

The Hospital Compare App leverages data from the [Hospital Compare][hc]
dataset from healthata.gov to explore how this open dataset could be
used for creating a recommendation system for hospitals.

Access the app at: https://hospital-compare.herokuapp.com/

The compressed dataset is available at: https://hospital-compare.herokuapp.com/static/Hospital_Revised_FlatFiles.zip

[hc]: https://data.medicare.gov/data/hospital-compare
